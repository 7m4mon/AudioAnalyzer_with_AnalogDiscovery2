Labview 2020 Community Edition 公開を記念して AnalogDiscovery2 を使用したオーディオアナライザーをLabview 2020を使って作ってみました。  
私の環境では振幅が小さくなると歪み率が悪化する傾向があり、    
（100mVp-pで2.4%程度）AD2の実力かなと思ったのですが、  
8903Bと比較すると、そうとも言い切れない感じです。  
オーディオ機器の測定にはそのままでは使えませんが、アナログ無線機の  
12dB SINAD感度くらいは測れるのではないでしょうか。
